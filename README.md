一个用于将(古)希腊语映射为中文的工具，与翻译软件不同，每个音节都有唯一对应的中文音译。

单元音（7个）：α，ε，η，ι，ο，υ，ω；
双元音（8个）：αι，αυ，ει，ευ，οι，ου，υι， ηυ；（来源：古希腊语简明教程-孙周兴）
辅音（17个）：β，γ，δ，ζ，θ，κ，λ，μ，ν，ξ，π，ρ，σ/ς，τ，φ，χ，ψ。

音节划分规则（‘v’代表单元音或双元音，‘c’代表辅音,辅音的长度为0-2）：
(...vc)(cvc)...(cvc)(cv...)

基础使用方法：
1.输入希腊文和中文，点击‘+’；
2.如果有多种音节划分方式，选择一种；
3.如果程序无法自动匹配，选择音节对应的中文；
4.确认后程序会保存（单词，音节列表）和（音节，发音）。

进阶使用方法：
1.如果已经保存了大量音节，输入希腊文，点击‘+’；
2.如果有多种音节划分方式，选择一种；
3.程序自动匹配中文；
4.确认后程序会保存（单词，音节列表）。

此项目使用Electron，Vite，Vue 3和Naive UI开发

A tool for mapping (Ancient) Greek to Chinese. Unlike translation software, each syllable has a unique corresponding Chinese transliteration.

Monophonic vowels (7): α, ε, η, ι, ο, υ, ω;
Diphthongs (8): αι, αυ, ει, ευ, οι, ου, υι, ηυ; (Source: 古希腊语简明教程-孙周兴)
Consonants (17): β, γ, δ, ζ, θ, κ, λ, μ, ν, ξ, π, ρ, σ/ς, τ, φ, χ, ψ.

Syllable division rules ('v' represents a monophthong or diphthong, 'c' represents a consonant, consonant length is 0-2):
(...vc)(cvc)...(cvc)(cv...)

Basic usage:
1.Enter Greek and Chinese characters, then click '+';
2.If multiple syllable division methods are available, select one;
3.If the program cannot automatically match, select the corresponding Chinese character for each syllable;
4.After confirmation, the program will save (word, syllable list) and (syllable, pronunciation).

Advanced usage:
1.If a large number of syllables have already been saved, enter Greek characters and click '+';
2.If multiple syllable division methods are available, select one;
3.The program will automatically match the Chinese character;
4.After confirmation, the program will save (word, syllable list).

This project is developed using Electron, Vite, Vue, and Naive UI.
